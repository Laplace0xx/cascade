# Cascade
Cascade is built or being built using Raylib, the concept is pretty interesting. You've got to survive against three enemy classes trolls, crawlers and swarms. The trolls are tall and slow , the crawlers are smaller and faster , the swarms are just flying creatures. When you destroy a troll, it splits into two crawlers , when you destroy a crawler it splits into a swarm and when you kill five creatures in a swarm, they turn to a troll. So you've got to use the terrain and tricks to survive for as long as you can

## Technologies
    This game is built using Raylib

## How to run
clone the repository

    git clone https://github.com/Laplace0xx/cascade.git
    cd cascade

Install Raylib [here](https://github.com/raysan5/raylib) if you haven't

Build with the Makefile

    make
    ./main
